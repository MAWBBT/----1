Контрольные вопросы

1. Пример структуры HTML5 документа:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Мой сайт</title> </head> <body> <h1>Заголовок страницы</h1> <p>Текст страницы.</p> </body> </html>

2. Новые семантические элементы в HTML5:

  • <article>: для обозначения отдельной статьи или раздела;
  • <aside>: для обозначения второстепенной информации;
  • <header>: для обозначения заголовка или шапки страницы;
  • <footer>: для обозначения нижнего колонтитула или подвала страницы;
  • <nav>: для обозначения навигации по странице.

3. Семантические элементы для текстового содержимого:

  • <p>: для обозначения параграфа;
  • <span>: для обозначения фрагмента текста;
  • <strong>: для выделения важного текста;
  • <em>: для подчёркивания текста.

4. Элементы для поддержки аудио, видео и подключаемых модулей:

  • <audio>: для воспроизведения аудио;
  • <video>: для воспроизведения видео;
  • <embed>: для встраивания внешних медиафайлов;
  • <object>: для встраивания интерактивных элементов.

5. Типы позиционирования:

  • static: элемент остаётся на своём месте и не перемещается при изменении размера окна браузера;
  • relative: элемент перемещается относительно своего исходного положения;
  • absolute: элемент перемещается относительно ближайшего родительского элемента с относительным позиционированием;
  • fixed: элемент закрепляется на экране и перемещается вместе с окном браузера.

6. Позиционирование внутри элемента:

  • left и right: перемещение элемента влево или вправо;
  • top и bottom: перемещение элемента вверх или вниз;
  • margin-left, margin-right, margin-top и margin-bottom: изменение расстояния между элементом и его границей.

7. Составные компоненты формы:

  • <form>: для создания формы;
  • <input>: для добавления полей ввода;
  • <label>: для указания названия поля;
  • <textarea>: для создания многострочного текстового поля;
  • <select>: для создания раскрывающегося списка;
  • <option>: для добавления опций в раскрывающийся список.

8. Глобальные атрибуты тега в HTML5:

  • accesskey: клавиша быстрого доступа к элементу;
  • class: определение класса CSS;
  • contenteditable: возможность редактирования содержимого элемента;
  • contextmenu: контекстное меню для элемента;
  • dir: направление текста в элементе;
  • draggable: возможность перетаскивания элемента;
  • hidden: скрытие элемента;
  • id: уникальный идентификатор элемента;
  • lang: язык элемента;
  • spellcheck: проверка правописания;
  • style: установка стиля элемента;
  • tabindex: порядок переключения элементов с помощью клавиши TAB;
  • title: дополнительное описание элемента;
  • translate: определение перевода содержимого элемента.
